# ManasMaster Elektrik

Сайт профессиональных услуг электрика в Алматы.

## О проекте
Это отдельный проект для электромонтажных работ, оптимизированный под SEO и готовый к деплою на Vercel.

## Технологии
- Vite
- React
- Tailwind CSS
- TypeScript

## Команды
- `npm install` — установка зависимостей
- `npm run dev` — запуск в режиме разработки
- `npm run build` — сборка проекта и генерация статических страниц
